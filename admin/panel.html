🏗 Архитектура

Система состоит из трёх компонентов:

  • Backend VPS — основной сервер. Здесь стоит 3x-ui, хранятся ключи, работает эта панель. IP backend знаешь только ты.
  • Relay VPS — дешёвый сервер-посредник ($3-5/мес). Принимает трафик от пользователей и перебрасывает на backend. Можно менять при блокировке.
  • Домены — указывают A-записью на relay. При блокировке relay меняешь только IP в DNS всех доменов.
Пользователь → domain.com → Relay VPS → Backend VPS

🚀 Быстрый старт

  1. Установи панель на backend VPS: pip3 install paramiko && python3 panel-server.py
  2. Открой в браузере: http://BACKEND_IP:7979
  3. Введи токен из /etc/mela/panel_token
  4. Перейди во вкладку Relay → добавь relay VPS
  5. Добавь свои домены в DNS (A-запись → IP relay)
  6. Добавь VPN ключи во вкладке Ключи
  7. Скопируй Subscription URL → вставь в приложение

🔄 Что делать если relay заблокировали

Это самый частый сценарий. Занимает 5-10 минут.
  1. Купи новый VPS ($3/мес на Hetzner, Vultr, DigitalOcean)
  2. Перейди во вкладку Relay
  3. Нажми Установить Relay — введи IP нового VPS и пароль
  4. Панель автоматически настроит проброс портов
  5. Обнови A-записи всех доменов → новый IP
  6. Готово. App начнёт работать как только DNS обновится (1-5 мин)
Backend IP не меняй никогда. Только relay.

🌐 Как добавить домен

  1. Купи домен (Namecheap, Porkbun, Cloudflare — от $1/год)
  2. В DNS панели регистратора добавь A-запись: @ → IP relay VPS
  3. Добавь домен в список зеркал (вкладка Зеркала)
  4. На каждом домене должен быть доступен /config.json — используй кнопку "Скопировать JSON"

🔑 Как добавить VPN ключи из 3x-ui

  1. Открой 3x-ui: ssh -L 2053:localhost:2053 root@BACKEND_IP → localhost:2053
  2. Создай inbound: VLESS + Reality, порт 443
  3. Скопируй ключ клиента
  4. Вставь во вкладке Ключи этой панели
  5. Нажми Сохранить — пользователи получат новый ключ при следующем обновлении подписки

📱 Настройка приложения

В файле lib/core/config/app_config.dart вставь свои домены:

static const seedUrls = <String>[ 'https://домен1.com/config.json', 'https://домен2.xyz/config.json', 'https://домен3.net/config.json', ];

Собери APK — больше менять не нужно. Всё управляется через эту панель.

🔒 Безопасность

  • Панель доступна по адресу http://VPS_IP:7979
  • Доступ защищён токеном (заголовок X-Token)
  • Токен хранится в /etc/mela/panel_token
  • Смена токена: echo "новый-токен" > /etc/mela/panel_token + перезапуск

⚙️ Запуск панели как сервис

cat > /etc/systemd/system/mela-panel.service << EOF [Unit] Description=MelaVPN Panel After=network.target [Service] ExecStart=/usr/bin/python3 /opt/mela/panel-server.py Restart=always RestartSec=5 [Install] WantedBy=multi-user.target EOF systemctl enable --now mela-panel
